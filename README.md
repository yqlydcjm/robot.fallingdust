# robot.fallingdust基于python-flask,go-cqhttp的qq机器人
### 使用方法
=======================================   
打开config.yml,更改第4，5行输入qq账号和密码   具体可以看go-cqhttp文档   
打开cmd输入go-cqhttp_windows_amd64.exe回车   
打开chatgpt.py输入你的openai的密钥   
打开main.py在第36行处输入你的qq号再运行即可   
### 内容
===========
私聊消息中可以实现群发功能具体命令为“群发 内容”，其他消息为chatgpt自动回复   
群聊消息中可以实现bmi指数计算："bmi 体重-kg 身高-m"，提问：“提问 内容”,表情捣："捣 qq号"，历史上的今天：”历史上的今天“
